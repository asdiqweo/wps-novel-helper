## 📦 下载和使用

### 安装方式

1. 克隆仓库
   ```bash
   git clone https://github.com/asdiqweo/wps-novel-helper.git
   cd wps-novel-helper
   npm install
   ```

2. 启动开发模式
   ```bash
   npm run dev
   ```

3. 在WPS中加载插件
   ```bash
   wpsjs debug
   ```

### 系统要求

- Node.js 16.x 或更高版本
- npm 或 yarn 包管理器
- WPS 专业版（支持插件开发）
